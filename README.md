# Confirmation of Payee

### Node.js version: ^22.14.0

Project setup:

- Install pnpm globally:

```sh
npm i -G pnpm
```

- Install dependencies using pnpm:

```sh
pnpm i
```

- Local development:

```sh
pnpm dev
```

- Run tests

```sh
pnpm test
```

- Build the production bundle

```sh
pnpm build
```

- Run the production build

```sh
pnpm preview
```
